Ask! Privacy Policy

Ask! automatically stores data when you interact with the bot. Below is listed what data we collect, and what we use it for. When you use the Ask! Discord bot, you automatically agree to this. Ask! saves information about you for moderation purposes and because the Ask! bot uses this data to make it work in the first place. All this data is publicly available, you can use the Ask! Question Viewer or directly read the data dictionary in the data.json file.

Question data

This is what Ask! saves about a question.

  • A question's Ask! ID

    Ask! uses a system with IDs. Ask! generates an ID upon the creation of the questions. The ID consists of two numbers (0-9) and three letter (A-z). An example of an ID would be 21AaJ. Automatically generated IDs contain three letters and two numbers, in a random order.

  • The question itself

    We store the question itself, so that this can be used by the Ask! bot and moderators. The Ask! bot appends any answers to the data.

  • The question history

    Whenever a question gets edited, the old question is saved.

    For example, the original question would be:

    How do I open this jar?

    And the author edits the question so it reads the following:

    How do I open this .jar file?

    The original question is stored automatically by the Ask! bot.

    If a question is not deleted, the question history is publicly available with the /history command. (if there is a history). Moderators can always view a question's history.

  • The author ID

    Every Discord account, including bots, has a unique ID. This ID can be used for moderation purposes, such as banning people from using the Ask! Discord bot.

    We store data about of question, which includes the question itself, its answers, who asked the Discord profile ID of the person who asked it (author ID), and the ID of the message. If the question is deleted, this also includes data about who deleted the question (a moderator of the author) and if the question is deleted in the first place.

  • The message ID

    This is the message ID of the message sent in the Ask! Discord server, this is used to edit the message when it gets deleted.

  • The author name and tag

    Every Discord account, including bots, has a unique name and tag. This tag is used by the Ask! Question Viewer to make it a lot more readable than, for example:

    Asked by 969293937091149874

  • Answer data

    We store data of answers of the questions. This is the answer itself and the author ID but also the message ID (in DMs, if DMs are closed, this is none) and if the question is the best answer.

User data

The things noted above are the things about questions. Ask! also stores the following user data. All of our user data is not private information, or things that should be kept private. Sometimes we don't even have user data at all.

  • User ID

    We store your ID, but only if you have executed a settings command, such as /toggle-dms.

  • User settings

    We store your settings, but only if you have changed them from their default value.

  • Ban status

    We store that you are banned. A reason is not included.

An example of how it looks is included below.

"618082876730376202": {
	"dms_enabled": "true",
	"banned": "false"
}

Note: All of this data is kept upon deletion, and if a moderator knows the ID, the entire question history can be viewed.
